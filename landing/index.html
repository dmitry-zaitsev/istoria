The log viewer for local dev.

Pipe any process into istoria and tail, search, and filter its logs in one window. Local, fast, no setup.

$ brew install dmitry-zaitsev/tap/istoria

what it does

  • Tails any process. Pipe stdout, attach to files, or follow Docker and Kubernetes pods.
  • Parses structured logs. JSON and logfmt fields become facets you can filter on.
  • Filter expression. level:error AND status_code:>=400.
  • Histogram. See spikes; click a bucket to scope the stream to that window.
  • Local index. Search runs on disk at ~/.istoria. No daemon, no server.
  • Stays local. No telemetry, no accounts, no outbound connections.

getting started

  1. Install via Homebrew. macOS and Linux.

    $ brew install dmitry-zaitsev/tap/istoria

  2. Pipe a process into it.

    $ npm run dev | istoria

  3. Level it up.

    • Chrome extension · stream a tab's console and network events into the same window.
    • MCP server · point local agents (Claude Code, Codex) at istoria to query logs in-session.
    • Alerts · save a filter, get a desktop notification when matches show up.
    • Saved views · pin filters as tabs and reopen them later.
